# 🏪 Smart Vendor Stall Portal (SVSP)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Node.js Version](https://img.shields.io/badge/node-%3E%3D%2018.0.0-brightgreen)](https://nodejs.org/)

**Smart Vendor Stall Portal (SVSP)** is a robust, centralized digital solution designed to streamline the management of vendor stalls in large event zones and marketplaces. It provides a seamless interface for vendor registration, real-time stall allocation, and automated attendance tracking.

---

## 🚀 Key Features

- **Digital Onboarding**: Automated registration workflow for new vendors with instant Employee ID generation.
- **Dynamic Stall Management**: Real-time status tracking of stall occupancy, equipment, and maintenance issues.
- **Self-Service Dashboard**: 
  - **Check-in/Out History**: Full visibility into past shift activities.
  - **Zone Navigation**: Easy viewing of available stalls in the assigned zone.
  - **Profile Management**: Secure access to verified identification documents and business details.
- **Administrative Transparency**: Clear maintenance logs and urgent support contacts.
- **Premium UI/UX**: Modern, responsive design optimized for both desktop and mobile operations.

## 🛠️ Technology Stack

- **Backend**: Node.js, Express.js
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Security**: Cookie-based session management, document encryption simulation.

## 📦 Installation & Setup

### Prerequisites
- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- npm (v9.0.0 or higher)

### Getting Started
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/Smart-Vendor-Stall-Portal.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd Smart-Vendor-Stall-Portal/Websit\ Files
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm start
   ```

The application will be accessible at `http://localhost:3000`.
